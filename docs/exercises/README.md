# Ejercicios prácticos — Academy E2E

Ejercicios para practicar la modificación del reporte HTML, la ejecución de los distintos tipos de reportes y la extensión del proyecto con nuevos Page Objects.

---

## Lista de ejercicios

| # | Ejercicio | Tema | Dificultad |
|---|-----------|------|-----------|
| 01 | [Personalizar el reporte CSS](01-CUSTOMIZE_REPORT_CSS.md) | `reports/academy_report.css` | Principiante |
| 02 | [Correr los reportes](02-RUNNING_REPORTS.md) | pytest-html vs Allure | Principiante |
| 03 | [Extender Page Objects](03-EXTEND_PAGE_OBJECTS.md) | Crear `InventoryPage` | Intermedio |

---

## Cómo usar estos ejercicios

1. Leer el ejercicio completo antes de empezar.
2. Intentar resolverlo por tu cuenta.
3. Si te trabás, la solución está al final de cada ejercicio bajo el título **Solución**.
4. Ejecutar los tests para verificar que todo funciona.

---

## Prerequisitos

- Python 3 con el entorno del proyecto instalado (`pip install -r requirements.txt`).
- Allure CLI instalado (opcional, solo para ejercicios 02 y 04).
- Editor de código con soporte CSS/Python.
